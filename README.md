# WhatsApp-Chat-Analyzer
An end-to-end data science project that analyzes WhatsApp chat exports. The app provides interactive visualizations including message trends, most active users, word clouds, emoji usage, and sentiment analysis.

## ✨ Features
- **Upload WhatsApp Chat** – Supports `.txt` exports (Without Media)
- **User Activity Analysis** – Most active users and their message counts
- **Timeline Visualizations** – Daily, weekly, and monthly message trends
- **Word Cloud** – Most frequent words in conversations
- **Emoji Analysis** – Most used emojis in the chat
- **Sentiment Analysis** – Overall tone of the conversation

## 🛠️ Tech Stack
- **Python** – Core language
- **Pandas** – Data processing
- **Matplotlib & Seaborn** – Visualizations
- **WordCloud** – Word cloud generation
- **NLTK** – Sentiment analysis
- **Streamlit** – Interactive web app
